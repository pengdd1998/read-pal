# Dr. Chen — Academic Researcher

## Profile
- **Age:** 34, PhD candidate in cognitive science
- **Reading habits:** 10+ papers/books per month, heavy annotator
- **Tech comfort:** High — uses Zotero, Mendeley, LaTeX
- **Motivation:** Research efficiency, knowledge management, cross-referencing
- **Patience:** Medium — tolerates complexity if it saves time long-term

## Simulation Tasks
1. **Multi-Book Upload**: Upload 3-5 academic texts. Test library organization and search.
2. **Deep Annotation**: Read a chapter, create 10+ annotations with notes. Test annotation panel.
3. **Cross-Reference Query**: Ask AI "How does this concept relate to what I read in [other book]?"
4. **Semantic Search**: Search across all books for "working memory capacity". Is it accurate?
5. **Export & Citation**: Try to export annotations with proper formatting. Can I cite these?

## Success Criteria (from Dr. Chen's perspective)
- Annotation system supports my workflow (highlight + note + tag)
- Search finds relevant passages across books, not just keyword matches
- AI provides academically rigorous responses with source awareness
- I can export my work in a useful format

## Failure Signals
- Annotations lost or disorganized
- Search only does keyword matching
- AI gives superficial "pop science" answers
- No export or citation functionality
