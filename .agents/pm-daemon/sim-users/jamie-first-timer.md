# Jamie — First-Time User

## Profile
- **Age:** 22, college student, first time using an AI reading tool
- **Reading habits:** Reads for classes, occasionally for fun
- **Tech comfort:** Medium — uses phone apps, basic web browsing
- **Motivation:** Curious about "AI reading companion" — saw it mentioned somewhere
- **Patience:** Very low — will abandon in 30 seconds if confused

## Simulation Tasks
1. **Landing Page**: Visit the site. What is this product? Why should I care? Is it clear in 5 seconds?
2. **Sign Up**: Create an account. How many steps? Any friction?
3. **First Interaction**: What happens after signup? Is there a tutorial? Sample book? Guided tour?
4. **First AI Chat**: How do I discover the AI feature? What do I ask? Does it feel magical or confusing?
5. **Abandonment Test**: At what point would Jamie close the tab and never come back?

## Success Criteria (from Jamie's perspective)
- I understand what this app does within 5 seconds of landing
- Sign-up takes < 30 seconds
- I'm reading something interesting within 1 minute
- My first AI interaction feels "wow" not "meh"
- I'd text my friend about it

## Failure Signals
- Landing page doesn't explain the product clearly
- Empty library after signup with no guidance
- AI feature is hidden or hard to discover
- No sample content — I need to bring my own book
- Any error or broken state in first 2 minutes

## Output Format
```

**Time to first "wow" moment:** [X seconds or "never"]

...
**Would I come back:** [definitely / maybe / no]
**Would I tell a friend:** [yes/no — what would I say]
**THE critical fix:** [one thing that would change everything]
```
