Adds read-pal, a self-hosted AI reading companion.

Features:
- Upload EPUBs and read with an AI companion that explains, asks questions, and connects ideas
- Personal knowledge graph connecting concepts across books
- Spaced repetition flashcards (SM-2 algorithm)
- Memory books — AI-generated reading journey compilations
- Book clubs, annotations, smart export (APA/MLA/Markdown/JSON)
- Full REST API with OpenAPI spec

Docker Compose deployment (PostgreSQL, Redis, FastAPI, Next.js, nginx). MIT licensed.

## General Submission Tips

1. **Search existing PRs** first — don't duplicate
2. **Follow the list's format** exactly (alphabetical, correct markdown)
3. **Keep PR description concise** — maintainers are busy
4. **Mention that it's MIT licensed** — important for most awesome lists
5. **Include demo link** when possible
6. **Star the repo** before submitting — shows genuine interest
7. **Be patient** — maintainers may take weeks to review
