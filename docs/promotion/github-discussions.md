# GitHub Discussions Setup Guide

## Enable Discussions

1. Go to https://github.com/pengdd1998/read-pal/settings
2. Scroll to "Features" section
3. Check "Discussions"

## Create Categories

Create these discussion categories:

| Category | Format | Description |
|----------|--------|-------------|
| Announcements | Announcement | Project updates and releases |
| General | Open-ended | General discussion about read-pal |
| Ideas | Open-ended | Feature requests and ideas |
| Q&A | Q&A | Questions and answers |
| Show and Tell | Open-ended | Share screenshots, reading stats, knowledge graphs |

## Welcome Post

Post this in **General** category:

---

### Welcome to read-pal! 👋

Thanks for checking out **read-pal** — an open-source AI reading companion that helps you read smarter, not harder.

**What does it do?**
- Chat with an AI companion about any book you're reading
- Build a personal knowledge graph that connects ideas across books
- Generate flashcards with spaced repetition
- Create beautiful "Memory Books" from your reading journey

**Quick Start:**
```bash
git clone https://github.com/pengdd1998/read-pal.git
cd read-pal
cp .env.example .env
docker compose up -d
```
Open http://localhost:8090 and start reading!

**How to contribute:**
See [CONTRIBUTING.md](https://github.com/pengdd1998/read-pal/blob/main/CONTRIBUTING.md) for guidelines. We'd love help with:
- Mobile responsiveness
- Browser extensions
- E-reader integrations
- Internationalization

**Share your experience:**
What are you reading? What features do you want? Drop a comment below!
